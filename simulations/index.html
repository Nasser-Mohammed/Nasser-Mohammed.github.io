Below are simulations I made and styled with HTML, CSS, and JavaScript. The core theme is modelling real life systems with ODEs and PDEs. I manually compute all integrals, derivatives, and differential equation updates. I do not use any mathametical libraries or imports other than the JavaScript math functions for rounding, random number generations, and the floor, trig, and exponential functions. Everything is built with the basic mathematical operations like +, -, *, /. I do use libraries for rendering LaTeX code and 3D visuals.
